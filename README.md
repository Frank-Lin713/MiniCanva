## 🚀 Features

- 🖊️ **Add Editable Text** — Create and style custom text blocks
- 🖼️ **Add Images** — Load placeholder images or customize with your own
- 📁 **Template Loader** — Load pre-designed templates to speed up design
- 💾 **Export Designs** — Download your creation as an image file
- ⚡ **Progressive Web App (PWA)** — Install and use offline
- 🧩 **Fabric.js Powered** — Uses Fabric.js for canvas rendering and object manipulation
